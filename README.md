# 💄 Makeup Analysis Dashboard

## 📌 Project Overview

This project analyzes makeup product sales data using Power BI.

The dashboard provides insights into:

- Total Sales
- Revenue Trends
- Product Performance
- Category Analysis
- Customer Buying Patterns
- Interactive Filters and Slicers

---

## 🛠 Tools Used

- Power BI
- Excel
- DAX
- Data Cleaning
- Data Visualization

---

## 📊 Dashboard Features

### Executive Overview
- KPI Cards
- Total Revenue
- Total Orders
- Average Sales

### Product Analysis
- Top Performing Products
- Category-wise Analysis
- Revenue Contribution

### Sales Trends
- Monthly Sales Trends
- Revenue Growth
- Performance Tracking
